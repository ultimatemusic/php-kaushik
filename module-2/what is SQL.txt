TCL :stand for transactional control language
	we can rollback using TCL
	
	a)commit
	b)rollback

	commit : commit is usedto save data before deleted
      START TRANSACTION;
      DELETE FROM users WHERE uid=4;
      COMMIT;
   rollback : rollback is used to recover data after delete
      START TRANSACTION;
      select * FROM users WHERE uid=4;
      COMMIT;

note: commit and roll back is not supported in MySQL
